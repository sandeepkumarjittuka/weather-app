# Weather App

A modern weather dashboard built using HTML, CSS, and JavaScript.

## Features

- Real-time weather search
- Temperature toggle (°C / °F)
- Weather icons
- Humidity information
- Wind speed information
- Feels Like temperature
- Local storage support
- Dynamic UI
- Error handling

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- GitHub
- Vercel
